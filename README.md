# Clone - Netflix Desktop App

    Node.js >12.22.0 
    Yarn

# Bibliotecas
    Storybook
    Jwt
    Jest
    Husky
    Keen-Slider
    StyledComponents
    next-iron-session
    axios
    Material-ui
    Context-API
    SWR

## Instalando o projeto

    yarn install

## Iniciando o projeto
<!-- desenvolvimento -->
    yarn dev 
<!-- build -->
    yarn build
    yarn start

Irá executar o app na porta padrão [http://localhost:3000](http://localhost:3000)

## Iniciando o storybook

    yarn storybook

Irá iniciar o storybook no padrão do app[http://localhost:6006/](http://localhost:6006/)
